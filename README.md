🧮 Calculadora

Uma calculadora simples desenvolvida com HTML, CSS e JavaScript.
O projeto foi criado com o objetivo de praticar lógica de programação, manipulação do DOM e eventos em JavaScript.

🚀 Funcionalidades

Inserção de números através dos botões

Operações matemáticas básicas:

➕ Adição

➖ Subtração

✖️ Multiplicação

➗ Divisão

Botão AC para limpar o display

Atualização dinâmica do display

Interface simples e intuitiva

🛠️ Tecnologias utilizadas

HTML5 → Estrutura da aplicação

CSS3 → Estilização da interface

JavaScript → Lógica da calculadora e manipulação do DOM
